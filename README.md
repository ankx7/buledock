# buledock

Posizionarsi nella directory /opt:
```ruby
cd /opt
```

Clonare da git:
```ruby
git clone https://github.com/ankx7/buledock.git
```

Entrare nella nuova dir:
```ruby
cd buledock
```

Rendere eseguibile il deploy.sh:
```ruby
chmod +x deploy.sh
```

Eseguirlo:
```ruby
./deploy.sh
```

Check in browser:
```ruby
http://<address>:8080
```
